# FitFound

**FitFound** is a next-generation, reverse-engineered job-hunting platform that redefines how hiring works. Instead of candidates endlessly applying to postings, FitFound empowers employers to discover and connect with top talent directly.

---

## 🚀 Tech Stack (MERN)

- **MongoDB** – NoSQL database  
- **Express.js** – Backend framework  
- **React.js** – Frontend UI library  
- **Node.js** – JavaScript runtime

---

## 🌟 Core Features

- ✅ Smart Candidate Matching  
- ✅ One-Time Universal Profile for Candidates  
- ✅ Direct Outreach & Interview Invitations  
- ✅ Verified Hiring Badges  
- ✅ Profile Visibility Insights
